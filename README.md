## Local Development

### Prerequisites

- Node (see [.nvmrc](./.nvmrc) for version)

### Run the project

```
$ git clone 
$ cd gatsby-netlify-cms
$ yarn
$ yarn develop
```

To test the CMS locally, you'll to need run a production build of the site:

```
$ yarn build
$ yarn serve
```
